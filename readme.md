📝 簡單介紹
這是一款專為小型團隊與部門打造的「零配置」雲端共用行事曆。無須繁瑣的註冊或登入流程，
只需輸入專屬的「部門代號」，即可瞬間建立獨立的協作空間。系統自動整合了台灣政府公告的國定假日與紅字，
並提供高彈性的自訂分類標籤（如：請假、交辦事項、會議）。
搭配極簡專業的 SaaS 介面與深色模式支援，無論在電腦或手機上，都能讓團隊成員的出缺勤與專案排程一目了然，實現最高效的資訊同步。

核心亮點：

即開即用：免帳號密碼，透過「部門代號」自動隔離與記憶您的專屬資料庫。

在地化整合：自動標示台灣例假日與國定假日名稱。

高辨識度介面：全天色塊與特定時段邊線設計，一秒掌握行程重點。

全端響應式：完美支援桌機與手機瀏覽，並內建一鍵深淺色護眼切換。
